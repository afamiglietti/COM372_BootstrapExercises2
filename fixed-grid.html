The Fixed Grid

All grid elements must be nested inside a container. We usually only use one container on a page, and we never nest containers inside each other. This page uses the 'container' class, which creates a fixed width design.

REMINDER: There are 12 columns in the grid system! So to use the full width of the container, your columns should sum to 12!

The "row" class contains the columns. We usually don't put content in a row without nesting it inside a column first, because the row has a negative margin.

Column Width 4

This is a column with a width of four. It has padding to keep it from crowding the next column. It has a width of 33.3%, which makes it take up a third of it's container. It doesn't change size in the fixed-width display (except at responsive breakpoints) because the container class has a fixed pixel width.
